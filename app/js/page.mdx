## 소개

자바스크립트 공부 기록입니다. 교재로는 **Javascript-The-Definitive-Guide**를 사용합니다.

인터렉티브하게 공부 내용을 기록할 수 있도록 공부에 앞서 몇몇 컴포넌트들을 준비했습니다.

우선 출력값 확인을 위해 아래 컴포넌트을 사용합니다. 코드를 수정해보세요.

```js
// console
console.log('Hello, world!');
```

테스트 케이스를 제공하는 컴포넌트도 준비했습니다. 복습할 때 활용할 계획입니다. 아래 컴포넌트에서 add 함수를 완성해보세요.

```js
// test
export const add = (a, b) => 0;

// jest
import { add } from './index.js';

describe('add', () => {
  test('Commutative Law of Addition', () => {
    expect(add(1, 2)).toBe(add(2, 1));
  });

  test('Random value', () => {
    const a = Math.random();
    const b = Math.random();
    expect(add(a, b)).toBe(a + b);
  });
});
```
