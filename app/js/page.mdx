인터렉티브하게 공부 내용을 기록할 수 있도록 몇몇 컴포넌트들을 준비했습니다. 출력값 확인을 위해 아래와 같은 컴포넌트을 사용합니다. 코드를 수정해보세요.

```js
// console
console.log('Hello, world!');
```

복습을 위한 문제 풀이 컴포넌트도 준비했습니다. 아래 add 함수를 완성하고 테스트를 실행해보세요.

```js
// test
export const add = (a, b) => 0;

// jest
import { add } from './index.js';

describe('add', () => {
  test('Commutative Law of Addition', () => {
    expect(add(1, 2)).toBe(add(2, 1));
  });

  test('Random value', () => {
    const a = Math.random();
    const b = Math.random();
    expect(add(a, b)).toBe(a + b);
  });
});
```
